# THIRD PARTY LICENSES

CyberVPN incorporates open-source software and frameworks to deliver strong privacy infrastructure.

## Core Libraries (Rust)
* **Tokio** (MIT) - Async Runtime
* **Reqwest** (Apache-2.0 / MIT) - HTTP Request Client
* **Serde** (MIT) - Serialization Engine
* **Tauri** (Apache-2.0 / MIT) - GUI Framework
* **Axum** (MIT) - Embedded Web Server Backend
* **rust-embed** (MIT) - Static Asset Embedding
* **uuid** (Apache-2.0 / MIT) - ID Generation
* **ring** (ISC) - Cryptographic Routines

## Frontend Libraries (React/NPM)
* **React** (MIT) - UI Library
* **Tailwind CSS** (MIT) - Styling Engine
* **Framer Motion** (MIT) - Cinematic Animations
* **React Three Fiber** (MIT) - 3D Scene Rendering
* **Lucide React** (ISC) - SVG Iconography
* **Zustand** (MIT) - State Management

*Compiled dynamically on build.*
