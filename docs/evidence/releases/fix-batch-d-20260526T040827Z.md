## Security Notes

- Magic-link and OTP 2FA branches intentionally return empty `access_token` and `refresh_token` fields and do not set auth cookies.
- Pending 2FA state is staged through the existing httpOnly frontend pending-2FA cookie flow.
- Logout clears local visible state immediately, but backend revocation remains authoritative for server-side token invalidation.
- Telegram linking retry no longer loops into the broken session-restore screen when the stored browser state is missing.

## Local Infra Notes
