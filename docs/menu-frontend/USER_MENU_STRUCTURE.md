## 1. Общее описание пользовательской системы

### 1.1 Целевые платформы

Пользователь VPN-сервиса может взаимодействовать с системой через следующие каналы:

| Платформа | Описание | Основные сценарии |
|-----------|----------|-------------------|
| **Мобильное приложение** | Flutter-приложение для iOS/Android | Быстрое подключение, выбор сервера, статус подписки |
| **Telegram-бот** | Основной канал продаж и поддержки | Покупка подписки, получение конфига, поддержка |
| **Веб-портал** | Customer-facing сайт на Next.js | Управление подпиской, скачивание клиентов |
| **Страница подписки** | Отдельная landing-страница | Покупка подписки без регистрации |

### 1.2 Статусы пользователя

| Статус | Описание | Ограничения |
|--------|----------|-------------|
| **active** | Активная подписка | Полный доступ к VPN |
| **trial** | Пробный период (2 дня, 2 ГБ) | Ограниченный функционал |
| **limited** | Исчерпан трафик | VPN не работает до продления |
| **expired** | Подписка истекла | VPN не работает |
| **disabled** | Заблокирован администратором | Полный доступ заблокирован |

### 2.2 Пункты главного меню

#### 2.2.1 Индикатор статуса подключения

**Отображаемая информация:**
- Текущий статус VPN-соединения (подключён/отключён)
- Название выбранного сервера
- Флаг страны сервера
- Время текущего подключения (при активном соединении)
- IP-адрес (при активном соединении)

**Возможные действия:**
- Tap по индикатору → быстрое переподключение
- Долгое нажатие → меню выбора сервера

#### 2.2.2 Кнопка «Подключиться» / «Отключиться»

**Состояния:**
- `default` → «Подключиться» (серый)
- `connecting` → анимация подключения
- `connected` → «Отключиться» (зелёный)
- `disconnecting` → анимация отключения
- `error` → «Повторить» (красный) + сообщение об ошибке

#### 2.2.3 Карточка статистики

**Отображаемые данные:**
- Использовано трафика в ГБ/ТБ
- Лимит трафика
- Прогресс-бар использования
- Осталось дней подписки
- Статус подписки (активна/истекает/истекла)

#### 2.2.4 Блок «Мои серверы»

**Функционал:**
- Отображение 3-5 избранных серверов
- Быстрый выбор сервера тапом
- Индикатор нагрузки сервера (низкая/средняя/высокая)
- Ping до сервера (мс)
- Статус сервера (онлайн/оффлайн)

#### 2.2.5 Карточка подписки

**Отображаемая информация:**
- Название тарифа (Basic/Pro/Ultra/Cyber)
- Дата окончания подписки
- Кнопка продления/апгрейда
- Информация о бонусах (если есть)

### 4.2 Фильтры и поиск

#### 4.2.1 Поиск сервера

**Функционал:**
- Поиск по названию сервера
- Поиск по стране (на английском или русском)
- Поиск по городу
- Поиск по коду сервера (DE1, US1, etc.)

**Голосовой поиск:**
- Кнопка микрофона
- Распознавание голоса

#### 4.2.2 Фильтр по региону

**Доступные регионы:**
- Все регионы
- Европа
- Америка
- Азия
- Ближний Восток
- Океания

#### 4.2.3 Дополнительные фильтры

| Фильтр | Опции | Описание |
|--------|-------|----------|
| **Статус** | Все / Онлайн / С обслуживанием | Показывать только работающие серверы |
| **Нагрузка** | Все / Низкая / Средняя / Высокая | Загрузка сервера |
| **Протокол** | Все / VLESS / XHTTP / WS-TLS | Поддерживаемые протоколы |
| **Ping** | Все / <20 мс / <50 мс / <100 мс | Время отклика |
| **Избранное** | Все / Только избранные | Серверы в избранном |

### 4.3 Карточка сервера

#### 4.3.1 Информация о сервере

| Поле | Описание |
|------|----------|
| **Название** | Код сервера + локация (DE1 - Frankfurt) |
| **Страна** | Флаг + название страны |
| **Город** | Название города |
| **Статус** | 🟢 Онлайн / 🔴 Офлайн / 🟡 Обслуживание |
| **Нагрузка** | Процент загрузки + визуальный индикатор |
| **Скорость** | Максимальная скорость порта |
| **Ping** | Время отклика в мс |
| **Онлайн** | Количество пользователей онлайн |

#### 4.3.2 Индикаторы статуса

```
🟢 Зелёный (низкая нагрузка <50%)    → Рекомендуется к использованию
🟡 Жёлтый (средняя нагрузка 50-80%) → Возможны замедления
🔴 Красный (высокая нагрузка >80%)  → Не рекомендуется
⚙️ Серый (обслуживание)             → Временно недоступен
```

#### 4.3.3 Поддерживаемые протоколы

Отображаются иконки доступных протоколов:

| Протокол | Иконка | Описание |
|----------|--------|----------|
| VLESS-Reality | ⚡ | Основной протокол, анти-DPI |
| XHTTP-Reality | 🔒 | Альтернативный протокол |
| WS-TLS | 🌐 | WebSocket + TLS |
| VMess | 📦 | Универсальный протокол |

#### 4.3.4 Действия с сервером

| Действие | Описание |
|----------|----------|
| **Tap** | Быстрое подключение |
| **Долгое нажатие** | Контекстное меню |
| **Свайп вправо** | Добавить в избранное |
| **Свайп влево** | Информация о сервере |

### 5.3 Подраздел «VPN-соединение»

#### 5.3.1 Автоподключение

**Опции:**
- **Отключено** → ручное подключение
- **При запуске приложения** → автоматически при открытии
- **При включении устройства** → автоподключение после перезагрузки
- **При потере соединения** → автоматическое переподключение

**Настройки интервала:**
- Интервал попыток: 5 / 10 / 30 секунд
- Максимум попыток: 3 / 5 / ∞

#### 5.3.2 Kill Switch

**Функционал:**
Блокировка интернет-трафика при разрыве VPN-соединения для предотвращения утечки данных.

**Режимы:**

| Режим | Описание | Использование |
|-------|----------|---------------|
| **Выключен** | Обычная работа | Базовый |
| **Блокировка интернета** | Полная блокировка при разрыве | Максимальная безопасность |
| **Блокировка приложений** | Блокировка только выбранных приложений | Гибкость |
| **Маршрутизация** | Блокировка определённых IP | Продвинутый |

**Исключения:**
- Возможность добавить доверенные приложения
- Список IP-адресов для исключения

#### 5.3.3 Выбор протокола

**Доступные протоколы:**

| Протокол | Преимущества | Недостатки | Рекомендация |
|----------|--------------|------------|--------------|
| **VLESS-Reality** | Максимальная скорость, анти-DPI | Требует SNI | Рекомендуется |
| **XHTTP-Reality** | Хорошая скорость, стабильность | Ограниченная поддержка | Альтернатива |
| **WS-TLS** | Работает везде | Низкая скорость | Для сложных блокировок |
| **VMess** | Универсальный | Средняя скорость | Универсальный |

**Автоматический выбор:**
- Анализ пинга и нагрузки
- Тест скорости до серверов
- Выбор оптимального протокола

#### 5.3.4 Настройки DNS

**Режимы DNS:**

| Режим | Описание | Преимущества |
|-------|----------|--------------|
| **Системный** | Использовать DNS системы | - |
| **VPN DNS** | DNS через VPN-туннель | Безопасность |
| **Кастомный** | Указать свой DNS | Гибкость |

**Популярные DNS:**

| DNS-провайдер | IP | Особенности |

| NextDNS | 45.90.28.0 | Кастомизация |
| CyberVPN | Внутренний | Без логов |

### Приложение C: Глоссарий терминов

| Термин | Определение |
|--------|-------------|
| **Kill Switch** | Функция блокировки интернета при разрыве VPN |
| **Split Tunneling** | Разделение трафика: часть через VPN, часть напрямую |
| **VLESS-Reality** | Протокол VPN с защитой от DPI |
| **XHTTP** | Альтернативный протокол для обхода блокировок |
| **WS-TLS** | WebSocket over TLS для маскировки трафика |
| **TOTP** | Time-based One-Time Password (двухфакторная аутентификация) |
| **SNI** | Server Name Indication (имя хоста в TLS-handshake) |
| **MTU** | Maximum Transmission Unit (максимальный размер пакета) |
