# Verification Commands

This file lists the commands Northstar expects to use as the repository grows.
Statuses mean:

- `active`: the command or wrapper exists now
- `planned`: expected once the Rust workspace exists
- `placeholder`: intentionally reserved until crate names or tooling choices are finalized
